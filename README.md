# fischr
Play games

# Installation

Install nodenv:


```
brew update
brew install nodenv
```

Install and use node 5.9.0:

```
nodenv install v5.9.0
nodenv local v5.9.0
node --version # make sure this shows 5.9.0
```

Install dependencies:

```
npm install
```

Run app:

```
node ./app.js
```

Visit http://localhost:3000/ in chrome
